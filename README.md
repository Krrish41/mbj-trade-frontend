# MBJ Global Pvt. Ltd. Website

Premium static B2B trading website for MBJ Global Pvt. Ltd., optimized for international buyers, importers, private label partners, and institutional buyers.

## Project Structure

- `index.html` - Single-page website markup
- `styles.css` - Responsive visual design and layout
- `script.js` - Tabs, product filters, sticky header, and inquiry form behavior

## Local Preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173/
```

## GitHub Pages

This project is GitHub Pages-ready with no build step.

1. Push the repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

The site will be published at:

```text
https://krrish41.github.io/mbj-trade-frontend/
```

## External Assets

The site uses Google Fonts, Lucide icons, and remotely hosted photography/map imagery. No package installation is required.
